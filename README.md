# NFT-Marketplace 

This marketplace is a semi-web3 NFT marketplace that uses metamask/crypto wallet as log-in option to access the website. It comes with a exlusive collection called DigBicks. 
<img width="1440" alt="Screen Shot 2023-01-27 at 9 39 01 AM" src="https://user-images.githubusercontent.com/107775878/215113467-477cb2d4-65f5-43b2-addc-fa0cc45128bc.png">

The collection is made up of cute 
pfps(picture for profile) diggies with different races,weapons, costumes, accessories and other styles. 

# Log-in 

<img width="601" alt="Screen Shot 2023-01-27 at 9 42 18 AM" src="https://user-images.githubusercontent.com/107775878/215113655-eb3b43d5-0413-4453-a147-568fa8a6cb87.png">

In order to view the collection, users need to use Metamask and connect their wallet to the web page. This is a cool web3 feature that is used in the modern day crypto-related dApps and Apps. 

# Create your NFT

<img width="543" alt="Screen Shot 2023-01-27 at 9 45 10 AM" src="https://user-images.githubusercontent.com/107775878/215114279-a3434f86-70cc-4547-adc8-008e4c8cbce7.png">


Users can add NFT in the marketplace by submitting the form. The NFT will automatically be included in the marketplace list after submission. This is a post request so it is persistent in the backend. You can grab NFTs from https://opensea.io/ by copying the image address and pasting it as the image source in the form. 

# Submission

<img width="904" alt="Screen Shot 2023-01-27 at 9 45 30 AM" src="https://user-images.githubusercontent.com/107775878/215114347-62fa12fd-6566-4c2d-aa4b-39802ef9c218.png">

Upon submission, you'll be redirected to your profile page and see the NFT that you created along with the owner name, total number of NFTs and the total value of all your NFTs that is denominated in `FLATCOIN` (FC).
